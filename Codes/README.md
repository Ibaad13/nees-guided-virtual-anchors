# Code — withheld pending publication

The implementation for this project is not included in this repository yet.

I'm currently writing up this work as a paper. I'm holding the source code back
until that process is further along / the paper is submitted or posted as a

*implementation*, not the description of the work.

If you're a professor, reviewer, or collaborator and want to see the code
ahead of the public release, reach out — contact details are in the main
README.
